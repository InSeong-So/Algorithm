# 무엇을?
손님에게 거슬러주어야 하는 거스름돈의 최소 개수 구하기

<br>

# 어떻게?
> 작은 돈부터 계산을 시작하면... 가령 1원부터 시작한다면 물건의 가격이 1원일 때 999원을 거슬러준다.
> - 그렇다면 큰 돈부터 거슬러 준다.

<br>

# 의사코드
```
N = 주어진 금액
count = 거스름돈의 개수
type = 동전의 유형

for x in 동전유형
  count에 주어진 금액에서 순회한 x를 전부 제거한 값 합산
  (N을 x로 나누어 몫을 계산)
  N = 주어진 금액에서 x를 제거한 나머지 값으로 재할당

return count
```
