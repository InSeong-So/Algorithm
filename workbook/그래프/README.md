# 그래프(Graph)
1. 정점(Node)과 간선(Vertex)으로 이루어진 자료구조
2. 일반적으로 인접행렬과 인접리스트로 표현한다.
3. 연결되어 있지 않은 노드는 임의의 수가 아닌 무한(Infinity)로 초기화해야 한다.

<br>

# 주요 알고리즘
1. DFS
2. BFS
3. 벨먼포드
4. 다익스트라
5. 플로이드-워셜
