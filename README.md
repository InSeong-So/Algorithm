# 알고리즘과 코딩 테스트 훈련

## 개요
- 알고리즘(풀이법) : 프로그램 개발에 있어 필요한 문제를 해결하는 방법을 체계적으로 정리하는 방법
  - 프로그램의 가장 작은 단위 : `입력(Input) - 처리(Process) - 출력(Output)`
    - 입력 : 자료 구조(Data Structure)가 담당하는 영역
    - 처리 : 알고리즘이 담당하는 영역
    - 출력 : UI가 담당하는 영역(콘솔, 데스크탑, 웹, 모바일 등)으로 가공된 데이터가 출력
